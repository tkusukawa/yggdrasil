Yggdrasil

Yggdrasil is a configuration management tool by Subversion.

Installation

Add this line to your application's Gemfile:

gem 'yggdrasil'

And then execute:

$ bundle

Or install it yourself as:

$ gem install yggdrasil

Usage

$ yggdrasil help

Environment

  • Linux
  • Subversion command-line client
  • Ruby
  • Gem
